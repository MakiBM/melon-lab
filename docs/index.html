toProcessable

Takes a human readable quantity and makes it processable by EVM according to the specified token decimals.

Note that the EVM always consumes and returns BigNumbers.

Eg: toProcessable(0.1, 'ETH-T')) --> BigNumber(100000000000000000) // (17 zeros)

toProcessable
Parameters
quantity (any)
symbol (any = "ETH-T")

toReadable

Takes a quantity (number) from the EVM and makes it human readable according to the specified token decimals.

Note that the EVM always consumes and returns BigNumbers.

Eg: toReadable(new BigNumber(100000000000000000, 'ETH-T')) // (17 zeros) --> // BigNumber(0.1)

toReadable
Parameters
quantity (BigNumber) number coming from EVM without precision.
symbol (string = "ETH-T") the symbol of the asset. Eg "MLN-T"
