toProcessable

Takes a human readable quantity and makes it processable by EVM according to the decimals specified by tokenSymbol.

Note that the EVM always consumes and returns BigNumbers.

toProcessable
Parameters
quantity ((number | BigNumber))
tokenSymbol (string = "ETH-T")
Returns
BigNumber
Example
toProcessable(0.1, 'ETH-T'));
 // --> BigNumber(100000000000000000) // (17 zeros)

toReadable

Takes a quantity from the EVM and makes it human readable according to the decimals specified by tokenSymbol.

Note that the EVM always consumes and returns BigNumbers.

toReadable

tokenSymbol (string = "ETH-T")
Returns
BigNumber
