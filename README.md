# Getting started

## Prerequisites

- yarn
- node >= v8.11.1
- docker >= 18.0.2

## Quickstart

```bash
git checkout git@github.com:melonproject/frontend-lab.git
cd frontend-lab
yarn install
yarn run bootstrap
yarn run dev
```

## With Docker

```bash
make bootstrap
make start
```
